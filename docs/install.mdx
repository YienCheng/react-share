## Create React App
Create React App 是React官方推荐的React脚手架工具，利用这个工具可以快速的搭建一个React单页面应用。

### 搭建项目
```bash
# 从npm拉取最新的create-react-app并运行
npx create-react-app my-app

# 进入项目目录
cd my-app

# 运行项目
npm start
```
如果需要学习create-react-app更多知识，请点击[了解更多](https://www.html.cn/create-react-app/)。
